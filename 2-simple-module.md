# Simple Module
<!--codex ignore simple-->
Here's an example of a simple module set up.

```javascript
// simple.js
console.log('Hello, simple!');

// app1.js
require('./simple.js');
```

1. Create the two files above and then run `node app1.js` in the terminal. What do you see?
2. Update `simple.js` with some of your own code.
